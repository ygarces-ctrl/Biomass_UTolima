# 🌲 Plataforma Analítica para la Estimación de Biomasa Aérea - Tolima

Aplicación interactiva desarrollada en el marco de investigación de la **Universidad del Tolima**, diseñada para la estimación y pronóstico temporal de Biomasa Aérea (Mg) utilizando **Google AlphaEarth Satellite Embeddings**, topografía y algoritmos de Machine Learning en vivo.

**Investigador Principal:** Yeison Alberto Garcés Gómez
